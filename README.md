# Basic Calculator in Python

This is a simple Python program that performs basic mathematical operations:

- Addition
- Subtraction
- Multiplication
- Division

## How It Works
The program takes two numbers as input from the user and displays the result of all four operations.

## Features
- User input support
- Handles division by zero
- Beginner-friendly code

## How to Run
1. Make sure Python is installed.
2. Run the file using:
   python filename.py
3. Enter two numbers when prompted.

## Example
Enter the first number: 10  
Enter the second number: 5  

Addition: 15  
Subtraction: 5  
Multiplication: 50  
Division: 2.0

# Personalized Greeting Program in Python

This is a simple Python program that creates a personalized greeting message.

## How It Works
- The program asks the user to enter their first name.
- Then it asks for their last name.
- It combines both names and displays a welcome message.

## Features
- User input handling
- String concatenation
- Beginner-friendly Python code

## How to Run
1. Make sure Python is installed.
2. Run the program using:
   python filename.py
3. Enter your first and last name when prompted.

## Example
Enter your first name: John  
Enter your last name: Doe  

Hello, John Doe Welcome to the Python program.
